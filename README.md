
# Sprint 10 - Proyecto Final / Final Project

## Español

### Contexto y objetivo
Este proyecto final integra todos los conocimientos adquiridos durante el bootcamp. El objetivo es analizar cómo el uso de redes sociales influye en la productividad y el bienestar de las personas, utilizando técnicas de análisis de datos, visualización y presentación de resultados.

### Metodología
1. **Recolección de datos:** Se utiliza el dataset público "Social media vs productivity" de Kaggle, que contiene información sobre hábitos de uso de redes sociales, productividad percibida y real, niveles de estrés, horas de sueño, entre otros.
2. **Preprocesamiento:** Limpieza de datos, tratamiento de valores nulos y transformación de variables para facilitar el análisis.
3. **Análisis exploratorio:** Se exploran las relaciones entre el tiempo en redes sociales, la productividad, el estrés y el bienestar. Se utilizan visualizaciones para identificar patrones y correlaciones.
4. **Visualización y presentación:** Los resultados se presentan en un informe, una presentación en Power BI y un notebook de Jupyter con todo el proceso documentado.
5. **Conclusiones:** Se resumen los hallazgos principales y se proponen posibles recomendaciones o líneas de investigación futura.

### Estructura del proyecto
- **Informe:** Informe Proyecto final.pdf (análisis detallado y conclusiones)
- **Notebook:** Proyecto final Python.ipynb (preprocesamiento, análisis exploratorio y visualizaciones)
- **Presentación Power BI:** Power BI Redes Sociales vs Productividad y Bienestar (dashboard interactivo)
- **Conjunto de datos:** Social media vs productivity.csv
	- Mashayekhi, M. (2023). Social media vs productivity dataset [Conjunto de datos]. Kaggle.
	- https://www.kaggle.com/datasets/mahdimashayekhi/social-media-vs-productivity
- **Presentación PDF:** Redes sociales vs productividad y bienestar.pdf

### Resultados principales
- Se observa una correlación negativa entre el tiempo diario en redes sociales y la productividad real.
- El uso excesivo de redes sociales se asocia con mayores niveles de estrés y menor satisfacción laboral.
- Las personas que utilizan aplicaciones de enfoque o bienestar digital tienden a reportar mejores niveles de productividad y bienestar.
- El análisis sugiere la importancia de establecer límites en el uso de redes sociales para mejorar el rendimiento y la calidad de vida.

### Cómo usar este sprint
1. Abre el notebook para seguir el proceso de análisis paso a paso.
2. Consulta el informe y la presentación para ver los resultados y conclusiones.
3. Explora el dataset para comprender la fuente y estructura de los datos.

## English

### Context and Objective
This final project brings together all the skills acquired during the bootcamp. The goal is to analyze how social media usage influences productivity and well-being, using data analysis, visualization, and presentation techniques.

### Methodology
1. **Data Collection:** The public "Social media vs productivity" dataset from Kaggle is used, containing information on social media habits, perceived and actual productivity, stress levels, sleep hours, and more.
2. **Preprocessing:** Data cleaning, handling of missing values, and variable transformation to facilitate analysis.
3. **Exploratory Analysis:** Relationships between social media time, productivity, stress, and well-being are explored. Visualizations are used to identify patterns and correlations.
4. **Visualization and Presentation:** Results are presented in a detailed report, a Power BI dashboard, and a Jupyter notebook documenting the entire process.
5. **Conclusions:** Main findings are summarized and possible recommendations or future research directions are proposed.

### Project Structure
- **Report:** Informe Proyecto final.pdf (detailed analysis and conclusions)
- **Notebook:** Proyecto final Python.ipynb (preprocessing, exploratory analysis, and visualizations)
- **Power BI Presentation:** Power BI Redes Sociales vs Productividad y Bienestar (interactive dashboard)
- **Dataset:** Social media vs productivity.csv
	- Mashayekhi, M. (2023). Social media vs productivity dataset [Dataset]. Kaggle.
	- https://www.kaggle.com/datasets/mahdimashayekhi/social-media-vs-productivity
- **PDF Presentation:** Redes sociales vs productividad y bienestar.pdf

### Main Results
- A negative correlation is observed between daily social media time and actual productivity.
- Excessive social media use is associated with higher stress levels and lower job satisfaction.
- People who use focus or digital well-being apps tend to report better productivity and well-being.
- The analysis suggests the importance of setting limits on social media use to improve performance and quality of life.

### How to use this sprint
1. Open the notebook to follow the step-by-step analysis process.
2. Check the report and presentation for results and conclusions.
3. Explore the dataset to understand the data source and structure.
